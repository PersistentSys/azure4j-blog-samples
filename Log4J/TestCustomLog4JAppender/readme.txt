About:
=====
This is a Web application that demonstrates the use of custom appender to log messages to table storage. 

Prerequisites:
=============
Windows Azure Subscription.

Setup Instructions:
==================
Edit log4j.xml and provide values for "partitionKey", "tableName", "storageEndpoint", "accountName" and "accountKey" properties.
