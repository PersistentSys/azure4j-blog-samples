About:
=====
This sample demonstrate how to query SQL Azure using Hibernate.

Prerequisites:
=============
A SQLAzure with let's say Company database having Employee table with schema as below
   Column Name    Data type
   ===========    =========
   Id             int
   Name           varchar(20)
   Department     varchar(20)
     

Setup Instructions:
==================
Edit hibernate.cfg.xml and provide values for "connection.url", "connection.username"
and "connection.password" property
